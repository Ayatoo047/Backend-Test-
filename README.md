**Project: Simple E-commerce API**

## Project features

1. Creation of user
2. User login
3. Creation of Products by admin
4. Viewing of products by users
5. Adding of products to cart by users
6. Checking out of cart by users
7. Placing Orders
8. Verifying orders
9. Retrieving order history
